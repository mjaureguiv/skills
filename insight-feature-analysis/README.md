# Insight Feature Analysis Skill

## Purpose
This skill analyzes a set of customer insights, deduplicates them, and proposes actionable product features mapped to those insights. It is designed to help product managers quickly turn qualitative feedback into structured feature ideas.

## How it Works
- Accepts a list of raw insights (numbered or free-form)
- Extracts and deduplicates unique insights
- Maps each insight or group of similar insights to a concrete feature
- Outputs a markdown table with insight summaries and feature proposals

## Output Example
See temp/insight_feature_table.md for a sample output.

## Usage
Place raw insights in a text file or provide them via prompt. Run the skill to generate the mapping table.
