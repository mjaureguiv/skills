# Insight-to-Feature Mapping for PB Licensing

| Insight ID(s) | Insight Summary | Feature Name | Feature Description |
|---------------|----------------|-------------------------------|------------------------------------------------------------------------------------------------------------------------------------------|
| 1, 10         | Need for license usage reports; Difficulty understanding current license utilization | License Usage Reporting Dashboard | Provide a dashboard displaying real-time and historical license usage, utilization trends, and breakdowns by user, team, and license type. |
| 2, 3          | Difficulty tracking assigned vs. available licenses; Confusion over license types and entitlements | License Inventory & Assignment Tracker | Offer a clear view of assigned vs. available licenses, with explanations of license types and entitlements. |
| 4, 5          | Request for automated license assignment/removal; Need for alerts on license expiration or overuse | Automated License Lifecycle Management | Enable automated assignment and removal of licenses, with configurable alerts for expiration or overuse. |
| 6             | Difficulty managing external/temporary users | External & Temporary User Management | Add dedicated tools for managing external and temporary users, including streamlined onboarding/offboarding. |
| 7             | Request for integration with HR or identity systems | HR/Identity System Integration | Integrate with HR and identity management systems to sync user status and automate license provisioning. |
| 8             | Lack of self-service for user/license requests | Self-Service License Request Portal | Allow users to request, renew, or release licenses through a self-service portal, reducing admin workload. |
| 9             | Need for audit logs of license changes | License Change Audit Logging | Maintain detailed audit logs of all license assignments, removals, and changes for compliance and review. |
