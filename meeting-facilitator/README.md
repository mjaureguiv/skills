# Meeting Facilitator

Find the right method to solve problems in meetings using design thinking, facilitation, and collaboration techniques.

## What This Skill Does

- Recommends facilitation methods for your meeting goals
- Provides step-by-step facilitation guides
- Suggests design thinking exercises
- Helps structure workshops and retrospectives

## How to Use

Open Copilot Chat (`Ctrl+Shift+I`) and ask:

```
What facilitation method should I use for [your goal]?

Help me plan a workshop to [objective]

What design thinking exercise fits this challenge?
```

### Example Prompts

```
I need to prioritize features with my team - what method should we use?

Plan a 2-hour workshop to generate ideas for our new product

What's a good retrospective format for a team that's frustrated?

How do I facilitate a decision-making session with stakeholders who disagree?
```

## Method Categories

| Category | Use For |
|----------|---------|
| **Ideation** | Generating new ideas (brainstorming, crazy 8s, SCAMPER) |
| **Prioritization** | Ranking options (dot voting, impact/effort, MoSCoW) |
| **Decision Making** | Reaching consensus (fist of five, consent, RAPID) |
| **Problem Definition** | Understanding issues (5 whys, fishbone, problem statement) |
| **Retrospectives** | Team reflection (start/stop/continue, sailboat, 4Ls) |

## Tips

- State your meeting goal clearly
- Mention time constraints
- Share participant count and dynamics
