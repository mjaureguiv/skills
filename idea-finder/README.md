# IdeaFinder

> Search skills and ideas by problem, domain, or question — and explore adjacent concepts through an interactive taxonomy graph.

## Quick Start

```
#IdeaFinder <your query>
```

## Web App

Open **IdeaFinder** tab at [http://localhost:3000](http://localhost:3000)

## What It Does

- **Semantic search** across all skills and scenarios (keyword + taxonomy scoring)
- **Exploration graph** — clickable chips grouped by Domain, Persona, Mechanism, Related Problem, and Search Phrases
- **Match explanations** — each result explains why it was found (keyword hits, taxonomy overlap)
- **Follow-up prompt** — asks what problem you were solving and offers to trigger IdeaFrame
