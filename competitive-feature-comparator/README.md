# Competitive Feature Comparator

Compare product capabilities against competitors in a structured way.

## Purpose

This skill helps Product Managers evaluate how their product compares to competitors across specific features or capabilities.

## When to Use

Use this skill when:

- Running competitive analysis
- Preparing for strategy discussions
- Identifying feature gaps
- Supporting roadmap decisions

## Example

Input:

"Compare Signavio vs Celonis for process simulation capabilities"

Output:

- Feature comparison table
- Strengths and weaknesses
- Identified gaps
- Strategic opportunities

## Example Output

### Input
Compare Signavio vs Celonis for process mining

### Output
- Signavio: strong in governance and modeling
- Celonis: strong in execution and analytics
- Gap: advanced simulation capabilities
- Opportunity: improve predictive insights
