# Marty Cagan PM Coach

AI-powered coaching based on Marty Cagan's Silicon Valley Product Group (SVPG) methodologies.

## What This Does

Get product management coaching using frameworks from INSPIRED, EMPOWERED, and TRANSFORMED - Marty Cagan's influential books on building products customers love.

## When to Use

- You want advice on PM best practices
- You're struggling with stakeholder management
- Your team feels like "feature factory"
- You need help with product discovery
- You're leading a transformation to empowered teams
- You want to grow as a product manager

## How to Use

Just describe your situation:

```
I'm struggling to get stakeholders to stop dictating features. How do I push back?
```

```

```

```
How do I become a better product manager?
```

```
What's the difference between a product team and a feature team?
```

## What You'll Get

- **Diagnosis** using SVPG frameworks
- **Direct, honest feedback** (Cagan-style candor)
- **Practical next steps** you can implement

## Key Frameworks Covered

| Framework | What It Is |
|-----------|------------|
| **The Four Big Risks** | Value, Usability, Feasibility, Viability |
| **Empowered vs Feature Teams** | Problems vs features, outcomes vs output |
| **Continuous Discovery** | Validate before you build |
| **Vision vs Strategy** | OKRs over roadmaps |
| **Missionaries vs Mercenaries** | Passion vs contract compliance |

## Core Principles

1. **Products over Projects** - Durable teams beat temporary ones
2. **Empowerment over Control** - Give problems, not features
3. **Discovery over Delivery** - Validate before building
4. **Outcomes over Output** - Business results matter
5. **Principles over Process** - Understand the "why"
