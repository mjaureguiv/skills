# getting-started

Your complete guide to the SAP Signavio PM Agent - what it can do and how to use it.

## What This Is

The SAP Signavio PM Agent is an AI-powered Product Manager assistant that automates the tedious parts of PM work. It's a collection of 30+ skills that can:

- **Create documents**: PowerPoint, Word, Excel, PRDs, Jira tickets
- **Process meetings**: Turn transcripts into notes, extract action items
- **Write content**: LinkedIn posts, release announcements, email replies
- **Coach you**: PM strategy, frameworks, prioritization, goal setting
- **Analyze data**: Customer feedback, requirements, market research

## Quick Start

### Setup (5 minutes)

**Recommended**: Follow the [Claude Code setup guide](https://pages.github.tools.sap/hAIperspace/hai-docs/llm-proxy/recipes/claude/)

### Try It

Just describe what you need:

```
Create a 5-slide presentation about our Q1 roadmap

Turn this meeting transcript into structured notes

Write a PRD for a new authentication feature

Help me prioritize these features using ICE scoring
```

## What Can I Ask For?

| Want This | Say This |

| Meeting notes | "Turn this transcript into notes" |
