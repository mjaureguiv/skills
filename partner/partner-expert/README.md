# Partner Expert

Your go-to skill for all SAP Signavio technology partner questions.

## What It Does

Answers questions about:
- **Build/Buy/Partner strategy** — When should SAP build, buy, or partner?
- **Partner models** — SolEx vs Endorsed Apps vs SAP Store vs OEM
- **Partner onboarding** — How to become a Signavio tech partner, the funnel, PPWG process
- **Existing partners** — Current portfolio, status, key contacts
- **Links & resources** — Find the right document fast

## How to Use

```
#partner-expert
```

Then ask your question. Examples:

- "What's the difference between an Endorsed App and a Solution Extension?"
- "Walk me through how a new ISV partner gets onboarded at Signavio"
- "Which existing Signavio partners are active?"
- "What are the criteria for deciding Build vs Partner?"
- "Where can I find the PPWG nomination template?"

## Knowledge Base

The skill draws from a curated set of reference files in `references/`. These are populated incrementally as source documents are downloaded and fed.

See the [link registry](references/link-registry.md) for all ~50 source documents with freshness tracking.

## Keeping It Updated

Run `#partner-refresh` periodically to review which documents need updating.
