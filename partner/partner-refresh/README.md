# Partner Refresh

Keep the partner knowledge base current with periodic document reviews.

## How to Use

```
#partner-refresh
```

The skill will:
1. Show which documents are stale or overdue for review
2. Guide you through downloading and re-ingesting updated versions
3. Track what was refreshed and when

## Freshness Rules

- ⭐ [MUST] docs → flag at **60 days**
- All other docs → flag at **90 days**
- Deprecated docs → verify or remove
