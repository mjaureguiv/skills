## Tips for Keywords

- Use **3-5 keywords** per feature
- Include **synonyms** customers might use
- Include **problem-related** terms (not just solution terms)
- Example: For "Impact Analysis", include:
  - Solution terms: impact, analysis, visualization
  - Problem terms: affected, change, downstream, propagation
