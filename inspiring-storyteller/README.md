# Inspiring Storyteller

Transform slides and data into compelling leadership speeches that people remember.

## What It Does

Takes dry content — bullet points, slides, data — and turns it into a human speech that:
- Grabs attention from the first sentence
- Tells a story, not a list
- Frames challenges as discoveries
- Ends with a clear call to action

## How to Use

Share your content (slides, bullets, topic) and ask for a speech. Specify:
- Audience (leadership, PMs, all-hands)
- Length (90 seconds, 2 minutes, 3 minutes)
- Tone (exciting kick-off, thoughtful update, rallying cry)

### Example Prompts

- "Turn this slide into a 2-minute speech for leadership"
- "Create an inspiring intro for our initiative kick-off"
- "Make this data exciting for our QBR presentation"

## The Story Arc

Every speech follows this flow:

1. **Hook** — Grab attention
2. **Setup** — What we believed
3. **Discovery** — The insight
4. **Opportunity** — The positive reframe
5. **Action** — What we're doing
6. **Invitation** — Bring them along

## Key Principles

- Write like you speak — no bullet points
- Frame positively — discoveries, not problems
- Keep it simple — anyone should understand
- Make it human — if it sounds like a bot, rewrite

## Subskills

### Roadmap Update (`roadmap-update/`)

Prepare roadmap materials for leadership rounds. Transforms ProductBoard export data into:

- **Feature Summary Table** — PB links with value propositions and confidence levels
- **Changelog Slide** — Single slide summarizing monthly/quarterly roadmap changes
- **Confidence Summary** — Features grouped by confidence level (High/Medium/Low)

**Input**: ProductBoard CSV export (Release checks monthly/quarterly view)

**Use for**: Monthly leadership reviews, roadmap updates, deadline-driven deliverables
