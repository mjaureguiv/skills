# LinkedIn Posts

Create professional LinkedIn thought leadership posts.

## What This Skill Does

- Crafts engaging LinkedIn posts
- Follows professional communication best practices
- Optimizes for LinkedIn's algorithm and engagement
- Matches your personal voice and style

## How to Use

Open Copilot Chat (`Ctrl+Shift+I`) and use the prompt:

```
#create-linkedin-post Write a post about [your topic]
```

### Example Prompts

```
#create-linkedin-post Share insights about our AI product launch

#create-linkedin-post Write a thought leadership post about PM best practices

#create-linkedin-post Create a post announcing my new role
```

## What Gets Generated

Posts are saved to:
```
outputs/linkedin-exports/linkedin-[topic]-[date].md
```

## LinkedIn Best Practices

- Hook in the first line
- Use line breaks for readability
- Include a clear call-to-action
- 1-3 relevant hashtags
- Optimal length: 150-300 words

## Tips

- Mention your angle (personal experience, data, opinion)
- Specify the tone (professional, casual, inspiring)
- Ask for variations: "Make it more personal" or "Add a question"
