# Word Document Generation

Create and manipulate Word documents.

## What This Skill Does

- Generates Word documents from content
- Creates formatted reports and documents
- Manipulates existing .docx files
- Applies templates and styling

## How to Use

Open Copilot Chat (`Ctrl+Shift+I`) and ask:

```
Create a Word document for [your content]

Generate a report in Word format

Format this as a professional Word document
```

### Example Prompts

```
Create a Word document with our product requirements

Generate a customer-facing proposal in Word format

Turn these meeting notes into a formatted Word report
```

## Prerequisites

- **Python** with `python-docx` library:
  ```bash
  pip install python-docx
  ```

## What Gets Generated

Word files are saved to:
```
outputs/[filename].docx
```

## Tips

- Describe the structure (sections, headers, bullets)
- Mention any branding requirements
- Reference SAP templates if applicable
