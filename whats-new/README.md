# What's New Announcements

Write user-facing release announcements and feature updates.

## What This Skill Does

- Creates professional What's New announcements
- Follows SAP communication guidelines
- Focuses on user benefits, not technical details
- Ready for release notes, in-app messaging, or emails

## How to Use

Open Copilot Chat (`Ctrl+Shift+I`) and use the prompt:

```
#write-whats-new Write an announcement for [your feature]
```

### Example Prompts

```
#write-whats-new Announce the new dashboard filtering capabilities

#write-whats-new Write a What's New for our Q1 release with these features: [list]

#write-whats-new Create an in-app message for the SSO migration
```

## What Gets Generated

Announcements are saved to:
```
outputs/wn-exports/whats-new-[feature]-[date].md
```

## Writing Guidelines

Good announcements:
- Lead with user benefit, not the feature
- Use simple language (no jargon)
- Include a clear call-to-action
- Keep it concise

## Tips

- Specify the audience (end users, admins, developers)
- Mention the release date if relevant
- Ask for variations: "Make it shorter" or "More formal tone"
