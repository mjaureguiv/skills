# PM Customer Requests Assistant

A skill for responding to customer inquiries with consistent, accurate answers. Organized by product.

## Products

| Product | Description |
|---------|-------------|
| [SPG](spg/) | SAP Signavio Process Governance customer support |

## How to Use

Invoke with `/pm-customer-requests-assistant` or paste a customer inquiry and ask for help.

Each product subfolder contains:
- Product-specific knowledge base
- FAQ and known issues
- Sample responses
- JIRA ticket history
