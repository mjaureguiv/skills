# Sample Responses: Feature Requests

Examples of good feature request responses for tone matching.

---

## Example 1: Multiple Feature Requests (Formal)

**Customer:** Enterprise customer, formal tone
**Request:** List of 5 feature requests

**Response:**
```
Dear [Name],

Thank you for taking the time to share these feature requests with us. We greatly appreciate the detailed feedback from [Company].

I have reviewed each item and logged them in our product backlog:

1. **Email CC functionality** - This is on our roadmap for Q3 2026
2. **Undo/Redo in editor** - Currently under evaluation
3. **Bulk case closure** - Added to our backlog for prioritization
4. **Custom dashboard widgets** - Planned for a future release
5. **API webhook support** - Already in development

Your input is invaluable in helping us prioritize our development efforts. I will keep you informed of any updates regarding these features.

Please don't hesitate to reach out if you have any questions.

Best regards,
[Name]
SPG Product Team
```

---

## Example 2: Single Feature Request (Casual)

**Customer:** Smaller customer, casual tone
**Request:** One specific feature

**Response:**
```
Hi [Name],

Thanks for the suggestion! The ability to [feature] is something we've heard from a few customers.

I've added your request to our backlog. While I can't promise a specific timeline, I'll make sure our PM team sees this feedback.

I'll let you know if this moves forward!

Cheers,
[Name]
```

## Example 3: Feature Already Planned

**Response:**
```
Great news! This feature is already on our roadmap. We're targeting [timeframe] for release.

I've noted your interest so we can keep you updated as we get closer to launch.
```

---

## Example 4: Feature Not Planned

**Response:**
```
Thanks for sharing this idea. While it's not currently on our roadmap, I've logged it in our feedback system.

We review customer requests regularly when planning future releases. The more customers who request a feature, the higher priority it gets.

I appreciate you taking the time to share your thoughts!
```
