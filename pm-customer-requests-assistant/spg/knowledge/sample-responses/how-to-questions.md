# Sample Responses: How-To Questions

Examples of good support responses for tone matching.

## Example 1: Step-by-Step Instructions

**Question:** How do I set up email notifications?

**Response:**
```
Hi [Name],

Here's how to set up email notifications in SPG:

1. Go to **Administration** > **Settings** > **Notifications**
2. Click **"Create Template"**
3. Configure your template:
   - Subject line
   - Body (Markdown supported)
   - Recipients
4. Save and activate

You can also set conditions for when notifications are sent (e.g., only on approval, only for specific case types).

Let me know if you run into any issues!

Best,
[Name]
```
