Current bugs, limitations, and workarounds.

## Active Issues

### Issue: Export to PDF fails for large diagrams
**Status:** Under investigation
**JIRA:** SPG-12345
**Affected versions:** All current
**Workaround:**
1. Export diagram in smaller sections
2. Or use PNG export instead
3. Or reduce diagram complexity before export

**ETA:** Q2 2026

---

### Issue: Email attachments limited to 10MB
**Status:** By design (system limitation)
**Workaround:**
- Use links to shared files instead of attachments
- Compress attachments before adding
- Split into multiple emails if necessary

---

### Issue: JavaScript errors in complex workflows don't show details
**Status:** Improvement planned
**Workaround:**
- Use try/catch blocks in JavaScript actions
- Log to console for debugging
- Test scripts in isolation first

**ETA:** Q3 2026
