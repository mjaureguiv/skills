# transcript-to-notes

Turn messy meeting recordings into clean, actionable notes in minutes.

## What This Skill Does

You give it a Microsoft Teams transcript (or any meeting recording text), and it creates organized notes with:

- **What was discussed** - Key topics grouped logically
- **What was decided** - Clear list of decisions made
- **Who does what** - Action items assigned to people
- **What's still open** - Follow-ups and unresolved questions

**In simple terms**: Instead of re-watching a meeting or reading through a messy transcript, you get a clean summary you can share with your team immediately.

---

## How to Use This Skill

Just give me the transcript. Here are some ways:

### Copy and paste it
```
Here's the transcript from today's sprint planning:

[paste the transcript text here]
```

### Point to a file
```
Create notes from this transcript: @downloads/meeting-recording.vtt
```

### Add some context
```
This was our weekly sync with the product team. Here's the transcript:

[paste transcript]
```

## What Happens Behind the Scenes

### Step 1: Getting Meeting Info (Optional)

If you have Outlook calendar connected, I can automatically:
- Find the meeting name from your calendar
- Get the attendee list
- Pull any links from the meeting invite

This is optional - the skill works fine with just the transcript.

### Step 2: Finding Who Said What

I scan through the transcript to identify:
- Who was in the meeting (from speaker tags)
- What each person discussed
- Any names mentioned

### Step 3: Creating the Notes

I organize everything into:
- A quick summary (2-3 sentences)
- Discussion topics (grouped by theme)
- Decisions made
- Action items (with names attached)
- Open questions and follow-ups

### Step 4: Saving the File

Notes are saved to:
```
meetings/2026-02-27 Sprint Planning.md
```

The filename uses the meeting name from your calendar, or I generate one from the content.

## Getting Transcripts from Microsoft Teams

### Option 1: Download the file
1. Open the meeting chat in Teams
2. Click on the transcript tab
3. Download as .vtt or .docx file
4. Give me the file path

### Option 2: Copy and paste
1. Open the transcript in Teams
2. Select all (Ctrl+A) and copy
3. Paste directly into our chat

---

## Connecting to Outlook Calendar (Optional)

This lets me automatically get meeting names and attendees. It's optional but makes filenames nicer.

**Setup:**
1. Go to: https://developer.microsoft.com/en-us/graph/graph-explorer
2. Sign in with your Microsoft account
3. Copy the access token
4. Set it as an environment variable

The token expires after about an hour, so you'll need to refresh it occasionally.

**Without calendar access:** Everything still works - I just generate a meeting name from the transcript content instead.

---

## If Something Goes Wrong

**No calendar access?**
No problem - I'll generate a meeting name from the content (e.g., "Sprint Planning with Sarah and Tom").

**Messy transcript with lots of "[inaudible]"?**
I'll note that quality is low and focus on what's clear. Unclear parts get marked as "[UNCLEAR: ...]".

**Really long meeting (1+ hours)?**
I'll group content by major topics and add a timeline if there are time markers.
